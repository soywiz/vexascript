# Processed Journal Notes

This directory stores engineering journal entries that have already been
ingested into actionable follow-up work.

Move a note here only after one of these is true:

- a new task was created from it
- an existing task was updated from it
- the note was reviewed and clearly determined to be already covered by
  existing tracked work

Processed notes remain useful historical context. Do not delete them just
because their lessons were turned into tasks.
